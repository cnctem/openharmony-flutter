# flutter_flutter

原始仓来源：https://github.com/flutter/flutter

## 仓库简介
本仓库包含开源软件Flutter，为OpenHarmony提供对Flutter的支持能力。

Flutter是一个跨平台框架，提供Flutter SDK和工具链，可支持IDE或者终端使用Flutter Tools指令编译和构建OpenHarmony应用程序。

## 使用文档
